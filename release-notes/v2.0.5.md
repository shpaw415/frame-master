# Frame-Master v2.0.5 Release Notes

## New Features

### Plugin CLI Extensions

- Introduce new CLI extension system for plugins using Commander.js
- Plugins can now extend the Frame-Master CLI with custom commands
- Access plugin commands via `frame-master extended-cli <plugin-command>`
- Comprehensive JSDoc documentation with practical examples for CLI development

### Build Environment

- Add `BUILD_MODE` environment variable when running `frame-master build`
- Helps distinguish between different build contexts in your application
- Prevent unwanted behaviors when only a build occurs

## Bug Fixes

- Fix prevent log functionality not properly preventing logs
- Fix request-handler not merging `body_init` correctly when `setResponse()` was called

## Other Changes

- Update Frame-Master logo
