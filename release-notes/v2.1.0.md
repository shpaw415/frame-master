# v2.1.0

## CLI Updates

### New `create` command

- **Interactive Mode**: Run `frame-master create` without arguments to launch an interactive wizard.
- **Template Support**:
  - Create projects from community templates: `frame-master create my-app --template cloudflare-react-tailwind`
  - Support for specific versions: `frame-master create my-app --template name@v1.0.0`
  - Automatic fetching from the Frame-Master Template Registry.
  - **Direct GitHub Support**: Create projects directly from a GitHub repository URL: `frame-master create my-app --template https://github.com/owner/repo`
  - **Automatic Cleanup**: Removes the project directory if template fetching fails (only if empty).

### Better Error Visibility

- `build` command will display more information when the build crashes.

### Global

- Added global verbose flag (`-v`, `--verbose`) for detailed logging.
